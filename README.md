# Desafio-ETL-na-Pr-tica---DIO-Criando-um-Pipeline-de-Dados-Resiliente-com-Python

Este projeto é um pipeline ETL desenvolvido em Python. Ele simula uma campanha de marketing bancário onde os dados dos clientes são analisados para gerar recomendações financeiras personalizadas.

Destaques Técnicos:

- Implementação de Mock Data (JSON local) para contornar indisponibilidade de APIs externas.

- Lógica condicional (Rule-Based) que personaliza mensagens baseadas em features financeiras (balance, limit), uma pequena simulação de IA.

- Uso de pandas, manipulação de JSON, List Comprehensions e tratamento de encoding UTF-8.

Estrutura: Código 100% modularizado, facilitando a futura integração com APIs REST reais e LLMs (OpenAI).
